### Prerequisites

## Getting Started

1. Open Terminal.
2. Navigate to your desired location to download the contents of this repository.
3. Copy and paste the following code into the Terminal :
   ```bash
   $ git clone git@github.com:RailonA/Discord-Telegram-Bridge-bot.git
   ```
4. Run `cd Discord-Telegram-Bridge-bot`
5. Install the needed dependencies 
    ```bash
    $ npm install
    ```
6. run
    ```bash
    node ./src/index.js
    ``` 

### Usage
Lauch the local version of this project go to the `src/index.js` file and run `node index`
